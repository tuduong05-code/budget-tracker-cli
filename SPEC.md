## Project Specification

### Goal

Create a simple command-line tool that allows college students and young adults to track personal finances by logging income and expenses, viewing summaries, and analyzing spending habits.

### MVP Features

1. Add a transaction (income or expense) with amount, category, and date
2. Save transactions to a CSV file
3. View all transactions
4. View a summary: total income, total expenses, and current balance
5. View spending grouped by category
